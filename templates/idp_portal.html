⚠️ Important: Install CA Certificate First

Before installing your personal certificate, you must download and install the CA certificate by clicking "Download CA Certificate" button above.

Windows

  1. First: Download and install the CA certificate (ca-certificate.crt)
  2. Download the .p12 certificate file
  3. Double-click the file to open Certificate Import Wizard
  4. Select "Current User" and click Next
  5. Enter password: certificate and click Next
  6. Select "Automatically select certificate store" and click Next
  7. Click Finish to complete the import

macOS

  1. First: Download and install the CA certificate (ca-certificate.crt)
  2. Download the .p12 certificate file
  3. Double-click the file to open Keychain Access
  4. Select "login" keychain
  5. Enter password: certificate and click OK
  6. The certificate will be added to your keychain

Linux

  1. First: Download and install the CA certificate (ca-certificate.crt)
  2. Download the .p12 certificate file
  3. Open terminal and run: pk12util -i certificate.p12 -d sql:$HOME/.pki/nssdb
  4. Enter password: certificate when prompted
  5. The certificate will be imported to your NSS database
