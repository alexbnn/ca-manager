Installation Instructions

×

Windows

  1. Download the .p12 certificate file
  2. Double-click the file to open Certificate Import Wizard
  3. Select "Current User" and click Next
  4. Leave the password field empty and click Next
  5. Select "Automatically select certificate store" and click Next
  6. Click Finish to complete the import

macOS

  1. Download the .p12 certificate file
  2. Double-click the file to open Keychain Access
  3. Select "login" keychain
  4. Leave the password field empty and click OK
  5. The certificate will be added to your keychain

Linux

  1. Download the .p12 certificate file
  2. Open terminal and run: pk12util -i certificate.p12 -d sql:$HOME/.pki/nssdb
  3. Press Enter when prompted for password
  4. The certificate will be imported to your NSS database
