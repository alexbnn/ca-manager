About SCEP Protocol

The Simple Certificate Enrollment Protocol (SCEP) is used by Apple devices to automatically request and receive digital certificates from a Certificate Authority.

SCEP Operations:

  • GetCACert: Retrieve the CA certificate
  • GetCACaps: Get CA capabilities and supported operations
  • PKIOperation: Submit certificate signing request (CSR)

Device Simulation:

This simulator creates realistic device profiles with proper:

  • Device model identifiers and serial numbers
  • Operating system versions
  • RSA key pair generation
  • Certificate Signing Request (CSR) creation
  • SCEP protocol interactions
